# Nexus ERP MVP

Projeto React + Vite pronto para deploy na Vercel.

## Rodar local
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Publicar na Vercel
### Opção 1: GitHub + Vercel
1. Suba esta pasta para um repositório no GitHub
2. Entre em https://vercel.com/new
3. Importe o repositório
4. Deploy

### Opção 2: CLI
```bash
npm i -g vercel
vercel
```

## Observação
Este pacote é um MVP funcional estruturado para deploy rápido.
